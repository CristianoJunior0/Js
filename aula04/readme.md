
    OPERADORES LOGICOS

        E (AND)
            Só é verdadeiro se ambas as partes forem verdadeiras
            &&
                true&&true === true
                true&& === false
                false&&true === false
                false&&false === false

        OU (OR)
            Só é falso se ambas as partes forem falsas          
            ||
                true||true === true
                true|| === true
                false||true === true
                false||false === false  

        NAO (NOT)
            Inverte o resultado (de verdadeiro para falso)        
            !
                !true === false
                !false === true
